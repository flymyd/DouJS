## 请求体示例

```json
{
  "type": 100,
  "data": {}
}
```

type为操作类型，定义如下：

* 100：设置昵称
* 101：创建房间
* 102：加入房间
* 103：退出房间
* 104：解散房间
* 105：列出房间
* 106：开始对局
* 201：查看手牌
* 202：出牌
* 203：叫分（TODO）
* 301：查看帮助
* 302：查看出牌规则
* 303：查看模式规则
* 304：查看随机事件
* 900：提权为管理员
* 901：重置所有数据（仅管理员可用）

## 返回体示例

```json
{
  "code": 0,
  "msg": "success",
  "type": 100,
  "data": {}
}
```

code定义如下：

* 0：错误
* 1：成功
